Perlindungan Data DNA & Medis

Selesai

Laporan implementasi keamanan web secara menyeluruh, merancang dan membangun sistem pertahanan berlapis untuk platform kesehatan fiktif "GeneCare". Proyek ini berfokus pada perlindungan data pengguna yang sangat sensitif, seperti data genetik (DNA) dan rekam medis, dari berbagai ancaman siber.

Jenis Proyek:Keamanan Aplikasi Web
Konteks:Laporan Ujian Akhir
Studi Kasus: Platform Kesehatan "GeneCare"
Teknologi Utama:PHP, HTTPS/TLS, Enkripsi AES-256, Hashing Password, Prepared Statements, Doppler
Pencapaian Utama:
  • Menerapkan enkripsi AES-256 untuk data DNA dan medis sensitif di database
  • Mengamankan autentikasi dengan Account Lockout, verifikasi Captcha, dan hashing password
  • Mencegah serangan SQL Injection menggunakan Prepared Statements
  • Membangun sistem registrasi aman dengan validasi input dan kebijakan password kuat
  • Mengamankan transmisi data dengan enkripsi HTTPS/TLS
  • Menerapkan manajemen kunci aman menggunakan layanan eksternal (Doppler)
